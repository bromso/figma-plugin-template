1. **Replace `package.json`** with the following exact content:

```json
{
  "name": "figma-plugin-template",
  "private": true,
  "packageManager": "bun@1.3.11",
  "workspaces": [
    "apps/*",
    "packages/*"
  ],
  "scripts": {
    "build": "turbo run build",
    "dev": "turbo run dev",
    "lint": "turbo run lint",
    "test": "turbo run test",
    "test:watch": "turbo run test:watch"
  },
  "devDependencies": {
    "turbo": "^2.9.5"
  }
}
```

Key decisions reflected:
- `"name"` changed from `"figma-plugin-react-vite"` to `"figma-plugin-template"` (matches repo name)
- `"version"` removed (private workspace root, not published)
- `"type": "module"` removed from root (each workspace package declares its own module type)
- All old scripts removed (turbo replaces npm-run-all chains)
- All old dependencies removed from root (they move to `apps/figma-plugin` in Phase 2)
- `"packageManager": "bun@1.3.11"` — NO `v` prefix (Turborepo rejects it)
- `"private": true` — required for Bun workspaces
- `"workspaces": ["apps/*", "packages/*"]` — Bun workspace globs (MONO-02)
- Only `turbo` as root devDependency

2. **Create `turbo.json`** with the following exact content:

```json
{
  "$schema": "https://turbo.build/schema.json",
  "tasks": {
    "build": {
      "dependsOn": ["^build"],
      "outputs": ["dist/**"]
    },
    "dev": {
      "cache": false,
      "persistent": true
    },
    "lint": {
      "outputs": []
    },
    "test": {
      "dependsOn": ["^build"],
      "outputs": ["coverage/**"]
    },
    "test:watch": {
      "cache": false,
      "persistent": true
    }
  }
}
```

Key decisions:
- Uses `tasks` key (not deprecated `pipeline`) — BUILD-01
- `build` and `test` have `"dependsOn": ["^build"]` for topological ordering — BUILD-02
- `dev` and `test:watch` have `cache: false, persistent: true` (long-running tasks)
- `lint` has `"outputs": []` (no cacheable output files)

3. **Create directory scaffolding:**
- Create `apps/.gitkeep` (empty file)
- Create `packages/.gitkeep` (empty file)

These ensure Git tracks the empty directories so `bun install` finds workspace roots on fresh clone.

4. **Update `.gitignore`** — append these entries (do not remove existing entries):

```
# Turborepo
.turbo
```

Do NOT add `bun.lock` to `.gitignore` — it must be committed.
