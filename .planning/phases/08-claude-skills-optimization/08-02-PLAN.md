- Common: packages/common/src/

- @repo/ui and @repo/common are JIT source-only packages (no build step)
- Import via workspace protocol: "workspace:*"
- Vite resolve.alias maps @repo/* to source directories

Keep existing content but update file paths to monorepo locations.

### Path Aliases
- @common -> packages/common/src
- @ui -> packages/ui/src
- @plugin -> apps/figma-plugin/src/plugin

### SVG Imports

## Tooling

### Biome
- Linting and formatting: `bun run lint` (via Turborepo)
- Config: biome.json (or biome.jsonc) at root
- Replaces ESLint + Prettier

### Vitest
- Test runner: `bun run test` (via Turborepo) or per-package `vitest run`
- Config: vitest.config.ts in each package that has tests
- packages/ui has @testing-library/react + happy-dom
- No test framework in apps/figma-plugin (plugin code untestable outside Figma)

### Turborepo
- Task definitions in turbo.json at root
- Tasks: build, dev, lint, test, test:watch, storybook, build-storybook
- dev and storybook are persistent (no cache)
- build and test depend on ^build (build dependencies first)
