1. Install rollup-plugin-visualizer in apps/figma-plugin:
   ```bash
   bun add -D rollup-plugin-visualizer --cwd apps/figma-plugin
   ```

2. Update `apps/figma-plugin/vite.config.ui.ts`:
   - Add import at top: `import { visualizer } from "rollup-plugin-visualizer";`
   - Add type import if needed: `import type { PluginOption } from "vite";`
   - Add env check before defineConfig: `const isAnalyze = process.env.ANALYZE === "true";`
   - Update plugins array to conditionally include visualizer and use filter(Boolean):
     ```typescript
     plugins: [
       react(),
       richSvg(),
       tailwindcss(),
       viteSingleFile(),
       isAnalyze && visualizer({
         open: true,
         gzipSize: true,
         template: "treemap",
         filename: "stats.html",
       }),
     ].filter(Boolean) as PluginOption[],
     ```

3. Add analyze script to `apps/figma-plugin/package.json` scripts:
   ```json
   "analyze": "ANALYZE=true vite build -c vite.config.ui.ts"
   ```

4. Add analyze task to `turbo.json` in the tasks object:
   ```json
   "analyze": {
     "dependsOn": ["^build"],
     "outputs": ["stats.html"],
     "cache": false
   }
   ```

5. Add analyze script to root `package.json` scripts:
   ```json
   "analyze": "turbo run analyze"
   ```

6. Verify the analyze command works end-to-end:
   ```bash
   bun run analyze
   ```
   This should produce `apps/figma-plugin/stats.html`.

7. Verify normal build does NOT produce stats.html:
   ```bash
   rm -f apps/figma-plugin/stats.html
   bun run build
   test ! -f apps/figma-plugin/stats.html
   ```
