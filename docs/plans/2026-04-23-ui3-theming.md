# Figma UI3 Theming Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Restyle all shadcn components to match Figma UI3's design language via CSS-only changes in `styles.css`.

**Architecture:** Update CSS custom property tokens for colors/radius/shadows, then add targeted `[data-slot]` selector overrides. Zero `.tsx` component file changes. Both light and dark themes.

**Tech Stack:** Tailwind CSS v4, OKLCH color space, CSS custom properties, `data-slot` attribute selectors
