## What is this?

This is a ready-to-use template for building Figma plugins. It includes everything you need: a modern tech stack, pre-built UI components, AI-powered development skills, and automated tooling.

**You don't need to know how to code.** Open this project in [Claude Code](https://claude.ai/code) and describe what you want your plugin to do. Claude will build it for you.

## What's included

- **14+ UI components** that look native in Figma (buttons, inputs, selects, accordions, and more)
- **AI skills** that understand Figma plugin architecture and can generate complete plugins from descriptions
- **Storybook** for browsing all components interactively
- **Automated testing** with 27+ tests
- **CI/CD pipeline** that validates every change
- **Changesets** for version management
